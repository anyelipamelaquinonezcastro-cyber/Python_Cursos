## 🗺️ ¿Qué vamos a construir en estas 10 sesiones?

**Oriente Stereo** es una emisora comunitaria del Oriente de Cali. Durante años han grabado entrevistas, crónicas y debates del territorio. Ese archivo sonoro es memoria colectiva — no tiene precio.

**El problema real:** parte de ese archivo ya se perdió por daño físico de equipos. El resto sigue en riesgo. Los audios están repartidos en tres lugares distintos (computador, discos externos y Google Drive), sin orden, sin nombres uniformes, sin respaldo sistemático.

**Nuestra misión en 10 sesiones:** construir un sistema en Python que:

| Sesión | ¿Qué construimos? |
|---|---|
| S1 | Entorno + primeras líneas de Python |
| S2 | Función para estandarizar nombres de archivos |
| S3 | Función para detectar archivos de audio |
| S4 | Script para explorar carpetas en múltiples rutas |
| S5 | Catálogo CSV con todos los programas |
| S6 | Módulo `organizer.py` con manejo de errores |
| S7 | Lógica de priorización del backup |
| S8 | Backup a Google Drive |
| S9 | Backup a AWS S3 (segunda capa) |
| S10 | Sistema completo + entrega a Oriente Stereo |
